GPA Calculator

Method of calculation: each grade has points as shown below we take each grade and multiply its points by the number of registered hours which is 3 and add the next subject in same way after adding all subject then you divide them by the number of subjects Multiplied by number of registered hours which is 3.

The points are:

A+ : 4 ===> Then A : 3.7 ===> Then B+ : 3.3 ===> Then B : 3 ===> Then C+ : 2.7 ===> Then C : 2.4 ===> Then D+ : 2.2 ===> Then D : 2 ===> Then F : 0.

Example to understand

if you have 6 subjects and its grade are A+, B, C+, C, D+, D then your GPA will be:

GPA = [ (4 * 3) + (3.3 * 3) + (2.7 * 3) + (2.4 * 3) + (2.2 * 3) + (2 * 3) ] / [ (6 * 3) ]
